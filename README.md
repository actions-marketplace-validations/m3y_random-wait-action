# random wait action

## Overview
This is an action to wait for a random number of seconds that does not exceed the specified maximum wait time.

## Inputs

### `max-wait-time`
**Required** Maximum waiting time. Default `10`.

## Outputs
No Outputs

## Example usage
```
uses: m3y/random-wait-action@v1
with:
  max-wait-time: 10
```
